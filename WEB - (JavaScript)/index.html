Triangle Solver

Відкрийте консоль браузера та викликайте функцію triangle(value1, type1, value2, type2).

Доступні типи: 'leg', 'hypotenuse', 'adjacent angle', 'opposite angle', 'angle'.

Приклади виклику функції:

triangle(4, "leg", 8, "hypotenuse");

triangle(8, "hypotenuse", 4, "leg");

triangle(30, "opposite angle", 5, "leg");

triangle(45, "angle", 10, "hypotenuse");

triangle(43.13, "angle", -2, "hypotenuse");
